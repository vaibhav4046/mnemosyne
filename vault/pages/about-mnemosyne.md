> Mnemosyne is a personal knowledge operating system.

Mnemosyne is a personal knowledge operating system. Unlike a generic note app, the LLM is the curator: it ingests sources, summarises them into atomic wiki pages, links them with [[wiki-slug]] cross references, and keeps the graph consistent. Three layers \ Collect, Connect, Recall \ match the Karpathy LLM Wiki pattern but layer in a 3D galaxy visualisation and a parallel agent swarm.
